    Papiro


    Built in 1991, Papiro is a table with oval top that takes the name of "unrolled paper pattern" of the cylinder that forms the base. Papiro was designed for the Fiam company.


    Model


    View code

    The model was made ​​entirely using the Bezier functions: for build the curves that outline the profile of the table, and to build their area.



    To the surfaces has been given thickness using the Extrude function.
